Inside the Code folder you will find the Installation Guide, User Manual, README.txt file and another folder labeled iOS
Application. Click on the iOS Application folder to find the source code for the application. From here, click on the Citizen
Scientist Project folder. Finally, click on the Citizen-Scientist-Project.xcodeproj file to open the application inside of
XCode and be able to execute it. To see the individual source files for the application you can continue through the Citizen
Scientist Project folder.
